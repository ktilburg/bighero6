# bighero6
Repo for the Big Hero 6 team in the interdisciplinary project.

# Getting Started
Requirements: Git and an IDE to view/edit the code.

The first step to start contributing is to clone this repository to a path on your own device using
`git clone https://github.com/ktilburg/bighero6.git`

To update the repository on your local device, checkout to the branch you want to view using
`git checkout [branchname]`

Then use
`git pull`
to retrieve the latest edits from the remote

For more information about what branches are you can visit:
https://www.w3schools.com/git/git_branch.asp
